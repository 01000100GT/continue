    Chat makes it easy to ask for help from an LLM without needing to leave the IDE. 
    
    You send it a task, including any relevant information, and it replies with the text / code most likely to complete the task. If it does not give you want you want, then you can send follow up messages to clarify and adjust its approach until the task is completed.
